# Naive_Bayes_Model_Evaluation

Overview
Naive Bayes Classification – NBA Player Longevity Prediction
Project Overview
In this project, you will analyze engineered NBA player data using Python and Scikit-learn to build a Gaussian Naive Bayes classification model. You will learn how to identify target variables, implement probabilistic classifiers, evaluate performance using confusion matrices and business-relevant metrics, and interpret model assumptions in the context of sports analytics. This project helps you translate statistical predictions into actionable scouting insights.

Step-by-Step Project Tasks
• Load the engineered dataset and confirm `target_5yrs` as the classification target
• Preprocess features to ensure compatibility with Gaussian Naive Bayes (continuous metrics)
• Split data into training and testing sets for unbiased model evaluation
• Implement a Gaussian Naive Bayes classifier using Scikit-learn
• Generate a Confusion Matrix to visualize true/false positives and negatives
• Calculate Precision (minimize false positives: "busts") and Recall (minimize false negatives: missed talent)
• Explain the Naive Bayes "Independence Assumption" and discuss its realism for basketball stats (e.g., correlation between points and minutes played)
• Summarize model reliability and limitations for a scouting department audience

Project Goal
By the end of this project, your analysis should:
• Successfully implement a Gaussian Naive Bayes model for binary classification
• Evaluate performance using Confusion Matrix, Precision, and Recall aligned with business priorities
• Critically assess the independence assumption and its impact on model validity in sports contexts
• Identify and document the most influential features driving longevity predictions
• Deliver a clear, stakeholder-ready summary explaining when to trust (or question) model outputs
• Provide actionable recommendations for integrating the model into scouting workflows
